# Point4Brand Assets Directory

This directory contains all visual assets for point4brand merchandise.

## Asset Organization

### Product Images
- `tshirt-*.jpg` - T-shirt product photos
- `mug-*.jpg` - Coffee mug product photos
- `tumbler-*.jpg` - Travel tumbler product photos
- `cup-*.jpg` - Specialty cup product photos
- `print-*.jpg` - Art print previews

### Collection Images
- `featured-*.jpg` - Collection hero images
- `hero-*.jpg` - Store hero section images

### Asset Naming Convention

Format: `{category}-{product}-{variant}.{extension}`

Examples:
- `tshirt-buddhabrot-black.jpg`
- `mug-lorenz-white.jpg`
- `print-collection-preview.jpg`

### Image Specifications

- **Product Photos**: 800x800px minimum, square aspect ratio
- **Hero Images**: 1200x600px, 2:1 aspect ratio
- **Collection Images**: 1000x800px, 5:4 aspect ratio
- **Format**: JPG for photos, PNG for graphics with transparency
- **Optimization**: Compressed for web delivery

### Keywords Integration

Each asset should be tagged with relevant keywords from the taxonomy:
- Mathematical concepts (fractal, chaos-theory, etc.)
- Visual themes (organic-patterns, network-visualization, etc.)
- Product attributes (premium, eco-friendly, etc.)
