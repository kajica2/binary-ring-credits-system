Quick Start

1

Get API Key

Register for a free API key to access generation and export features

2

Install SDK

npm install @binaryring/api-sdk
3

Generate Art

import { BinaryRingAPI } from '@binaryring/api-sdk';

const api = new BinaryRingAPI('your-api-key');

// Generate a Buddhabrot fractal
const result = await api.generate('buddhabrot', {
  iterations: 5000,
  samples: 10000000,
  zoom: 2.5
});

console.log('Generated:', result.imageUrl);

Authentication

The Binary Ring API uses API keys for authentication. Include your API key in the Authorization header:

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.binaryring.art/v1/projects
const headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
};

const response = await fetch('https://api.binaryring.art/v1/projects', {
  headers
});
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.binaryring.art/v1/projects', headers=headers)

Projects API

Access project information, parameters, and metadata for all 36 Binary Ring projects.

GET /v1/projects List all projects

Parameters

Parameter Type Description
category string Filter by category (fractals, attractors, particles, etc.)
featured boolean Show only featured projects
limit integer Maximum number of results (default: 20, max: 100)

Example Response

{
  "projects": [
    {
      "id": "buddhabrot",
      "title": "Buddhabrot",
      "description": "Fractal path tracing in the Mandelbrot set",
      "category": "fractals",
      "featured": true,
      "parameters": {
        "iterations": {
          "type": "integer",
          "min": 1000,
          "max": 10000,
          "default": 5000
        }
      },
      "api": {
        "generate": "/api/v1/generate/buddhabrot",
        "export": "/api/v1/export/buddhabrot"
      }
    }
  ],

  "page": 1,
  "limit": 20
}

Generation API

Generate artwork programmatically with customizable parameters for each project.

POST /v1/generate/{project_id} Create generative artwork

Request Body

{
  "parameters": {
    "iterations": 5000,
    "samples": 10000000,
    "zoom": 2.5,
    "centerX": -0.7,
    "centerY": 0
  },
  "format": "png",
  "resolution": {
    "width": 1920,
    "height": 1080
  },
  "async": true
}

Response

{
  "job_id": "gen_123456789",
  "status": "processing",
  "estimated_time": 30,
  "created_at": "2024-11-09T12:00:00Z",
  "parameters": {...},
  "webhook_url": "https://your-app.com/webhook"
}

Export API

Export generated artwork in various formats including high-resolution images and video animations.

POST /v1/export/{project_id}/{format} Export artwork in specific format

Supported Formats

  • png - High-quality PNG images (up to 8K)
  • jpg - Compressed JPEG images
  • svg - Vector graphics (where applicable)
  • webm - Video animations
  • gif - Animated GIFs
  • pdf - Print-ready PDFs

Request Body

{
  "parameters": {...},
  "export_options": {
    "resolution": "4k",
    "quality": 95,
    "animation": {
      "duration": 10,
      "fps": 30
    },
    "metadata": true
  }
}

SDK Examples

Generate Buddhabrot

const api = new BinaryRingAPI(process.env.BINARY_RING_API_KEY);

// Generate a custom Buddhabrot
const job = await api.generate('buddhabrot', {
  iterations: 8000,
  samples: 50000000,
  zoom: 5,
  centerX: -0.75,
  centerY: 0.1
});

// Wait for completion
const result = await api.waitForCompletion(job.job_id);
console.log('Artwork ready:', result.image_url);

Interactive Node Garden

// Create an interactive node garden
const garden = await api.generate('node.garden', {
  maxNodes: 200,
  connectionDistance: 150,
  growthRate: 0.03,
  interactive: true
});

// Get real-time updates
api.subscribeToUpdates(garden.job_id, (update) => {
  console.log('Network state:', update.nodeCount, 'nodes');
});

Export High-Resolution Art

// Export 8K version
const exportJob = await api.export('substrate', 'png', {
  resolution: '8k',
  parameters: {
    initialCracks: 8,
    growthRate: 1.5,
    spawnChance: 0.4
  }
});

const downloadUrl = await api.getExportResult(exportJob.job_id);
// downloadUrl contains the high-res image

Search and Filter

// Search for interactive fractal projects
const results = await api.search({
  query: 'fractal',
  filters: {
    category: 'fractals',
    interactive: true,
    difficulty: ['beginner', 'intermediate']
  },
  sort: 'popularity',
  limit: 10
});

results.projects.forEach(project => {
  console.log(`${project.title}: ${project.likes} likes`);
});
